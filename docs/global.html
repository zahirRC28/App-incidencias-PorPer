(async) conectar(urlApi, methodopt, bodyopt, tokenopt, responseTypeopt) → {Promise.<(Object|Blob|ArrayBuffer|Error)>}

Función helper para realizar llamadas HTTP a la API
Parameters:
Name Type Attributes Default Description
urlApi string URL del endpoint de la API
method string <optional>
 'GET' Método HTTP ('GET', 'POST', 'PUT', 'DELETE', etc.)
body Object | FormData <optional>
 {} Cuerpo de la petición. Puede ser un objeto JSON o FormData para archivos
token string <optional>
 Token JWT para autorización (opcional)
responseType 'json' | 'blob' | 'arrayBuffer' <optional>
 'json' Tipo de respuesta esperada - 'json': Devuelve respuesta parseada a JSON (default) - 'blob': Devuelve la respuesta como Blob (archivos binarios, imágenes, pdf) - 'arrayBuffer': Devuelve la respuesta como ArrayBuffer (para manipulación de bytes)
Source:
  • helpers/fetch.js, line 1
Returns:
- Retorna la respuesta de la API según `responseType`, o un objeto de error
Type
Promise.<(Object|Blob|ArrayBuffer|Error)>
Examples
// Llamada GET a la API con token
const datos = await conectar('/api/incidencias', 'GET', {}, miToken);
// Llamada POST con body JSON
const datos = await conectar('/api/incidencias', 'POST', { titulo: 'Prueba' }, miToken);
// Descargar archivo como Blob
const archivo = await conectar('/api/incidencias/1/pdf', 'GET', {}, miToken, 'blob');
